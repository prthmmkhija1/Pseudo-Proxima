# Proxima Agent Documentation

Welcome to the Proxima documentation. This site covers installing, configuring, and extending the Proxima agent for quantum backend comparison and LLM-assisted workflows.

## What is Proxima?
- Multi-backend quantum execution orchestrator
- LLM-assisted planning and reporting
- Export and comparison engine with rich CLI and TUI
